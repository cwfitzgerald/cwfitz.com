rend3

Github

Standalone renderer library designed to use the most modern gpu features. Fully gpu-based pipeline, using Descriptor Indexing, GPU Object Culling, and Clustered Forward Lighting.
My capstone/graduation project. Started in September 2020.

As of 2020-10-01, the following things are implemented:
  • Reverse Infinite Depth
  • Bindless Materials
  • GPU Frustum Culling
  • Use of Multi-Draw Indirect
The following things are in the works:
  • Clustered Forward Lighting w/ GPU Culling
  • Cascading Shadow Maps
  • Reflection/Global Illumination Probes
  • Physically Based Lighting
  • Screen Space Ambient Occlusion
  • Screen Space Reflections
  • Animation & Skinning

BVE-Reborn

Github

Open Source Train Simulator. Rewrite of OpenBVE in Rust and wgpu. Renderer features Order Independent Transparency, Clustered Forward Lighting, and GPU Light Culling.

Renderer features:
  • Reverse Infinite Depth
  • Order Independent Transparency
  • Clustered Forward Lighting
  • GPU Light Culling

wgpu

Github

Safe Rendering API in Rust. Serves as WebGPU implementation for both Firefox and Rust.
I've added extensions for Descriptor Indexing, Multi-Draw Indirect, and more. I've also worked on making the Rust experience better.
